Plusly

Een feature-rich Matrix client voor veilige, gedecentraliseerde communicatie

Download APK GitHub

Features

🔒

Veilige Messaging

End-to-end encryptie voor al je gesprekken. Jouw privacy staat voorop.

🌐

Gedecentraliseerd

Geen centrale server. Jij beheert je eigen data en communicatie.

🌍

Vertalingen

Real-time berichtvertaling met LibreTranslate. Communiceer in elke taal.

📱

Cross-Platform

Beschikbaar op Android, iOS, desktop en web. Overal en altijd.

🎨

Aanpasbaar

Personaliseer je ervaring met themes en instellingen naar jouw smaak.

💜

Open Source

Volledig transparant. Geen verborgen agenda's, geen data verkoop.
